# blog-command-tool

> 简单的nodejs命令行工具，类似hexo的静态博客工具


## usage

```bash
Usage: myblog [options] [command]


Commands:

  help                   显示使用帮助
  create [dir]           创建一个空的博客
  preview [dir]          实时预览
  build [options] [dir]  生成静态html

Options:

  -h, --help     output usage information
  -V, --version  output the version number

```
